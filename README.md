## Описание

TaskMaster — десктопное приложение для управления персональными задачами, разработанное на языке Python с графическим интерфейсом на PyQt5. Программа позволяет пользователю создавать структурированный список задач, классифицировать их по приоритету, устанавливать сроки выполнения (дедлайны), а также отслеживать статус выполнения.

В отличие от коммерческих аналогов (Todoist, Any.do, Microsoft To Do), программа имеет минималистичный интерфейс, не требует подключения к интернету и полностью контролируется пользователем. Все данные хранятся локально в формате JSON.

## Функциональные возможности

- Создание новой задачи с указанием названия (обязательно), описания, приоритета и дедлайна
- Просмотр полного списка задач с отображением статуса и сортировкой по приоритету
- Редактирование существующих задач с возможностью оставить поля без изменений
- Удаление задач с запросом подтверждения
- Отметка задач как выполненных с возможностью переключения статуса
- Поиск задач по ключевым словам в названии и описании (без учета регистра)
- Сортировка задач по приоритету: Высокий → Средний → Низкий
- Отображение статистики: всего задач, выполнено, ожидают, распределение по приоритетам
- Экспорт статистики в текстовый файл
- Автосохранение данных каждые 5 минут
- Сохранение и загрузка данных при запуске и закрытии программы
- Цветовое кодирование приоритетов и статусов задач

## Требования к системе

| Параметр | Требование |
|---|---|
| Операционная система | Linux (любой дистрибутив) |
| Процессор | x86_64, 1+ ГГц |
| Оперативная память | Не менее 512 МБ |
| Дисковое пространство | Не менее 50 МБ |
| Python | 3.8+ (для запуска из исходного кода) |
